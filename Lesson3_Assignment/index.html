Highlight #1 The Ping Program

"Ping" is a network utility that is used to determine the reachability of a host with an IP address. An IP address is the number assigned to a device part of any network that uses the Internet as a means of communication. The round-trip time it takes for a packet to reach an IP address and receive a reply is measured.

Highlight #2 Types of Webpages

Websites are designed with a variety of specifications but generally, they can be divided into two categories: static and dynamic. A static web page is composed of a series of HTML pages that can be viewed individually. Static webpages are the format that most sites were designed with in the early stages of the internet, before the development of complex and flexible design elements like those of dynamic webpages. Dynamic webpages implement the use of server technologies to build and display a webpage as soon as it's visited by a user. Furthermore, dyanamic webpages are typically regarded as the more functional form of web design because of the cohesive ability that having a single webpage provides a developer with.

Highlight #3 Encryption

When a developer intends to convert data into a language that requires a certain key or password, they encrypt it. Encryption is a tool designed so that data is made private in a way that is only accessible to authorized parties. This conversion of data into ciphertext can be achieved one of two ways: symmetrically or asymmetrically.
